## Sobre este proyecto

Bienvenido a mi portafolio. Este repositorio es una bitácora de mi aprendizaje en desarrollo web.  
A través de 10 ejercicios prácticos, documento mi evolución desde una simple página en blanco hasta interfaces interactivas que realizan cálculos en tiempo real.

El enfoque principal es escribir **código limpio**, usar **semántica correcta** y garantizar una buena **experiencia de usuario**.

---

## Tecnologías utilizadas

- **HTML5** – Estructura semántica y accesible.
- **CSS3** – Variables, Flexbox, selectores avanzados y hojas externas.
- **JavaScript (Vanilla)** – Manipulación del DOM, eventos y validaciones.
- **SweetAlert2** (Ejercicio 10) – Alertas modernas en lugar de las nativas.
- **Git y GitHub** – Control de versiones y publicación.

## Notas de la actividad

Durante esta práctica reforcé conceptos clave de programación web.  
Descubrí nuevas posibilidades como los selectores CSS avanzados, la validación de formularios en el lado del cliente y la integración de JavaScript para manipular el DOM.

El mayor reto fue entender cómo CSS trata los elementos como “cajas” y cómo aplicar selectores de pseudo-clase para mejorar la interfaz.
